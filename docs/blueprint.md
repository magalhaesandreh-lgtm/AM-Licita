# **App Name**: AM Licitações - MVP

## Core Features:

- App Shell: Basic layout with Sidebar and Header, navigation, and responsive design.
- Route Management: Placeholder routes for all key pages: /login, /dashboard, /certames, etc.
- Local Storage Service: Abstraction layer for LocalStorage with get, set, remove, clear methods.
- Data Table Component: Reusable data table component with sticky header, hover, sorting and EmptyState.
- Form Dialog Component: Modal form component with open/close management, scrollable body and standard buttons.
- Playground Repository: Example data repository using LocalStorage. Implemented as an example to show how this can be extended to connect to Firestore.
- Playground Screen: A test screen to validate the inputs and their persistence to localStorage.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) for a professional and reliable feel.
- Background color: Very light gray (#F5F5F5) for a clean backdrop.
- Accent color: Rich gold (#D4AC0D) as a highlight for actionable items.
- Body and headline font: 'Inter' sans-serif for modern readability.
- Cards with soft shadows and rounded borders (12-16px radius) to soften the interface.
- Consistent spacing using p-4 and p-6 for a structured layout.
- Simple, consistent icons for navigation items and actions within the data tables.
